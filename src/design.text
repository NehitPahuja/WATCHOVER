# 📘 WatchOver – Product Design Document

## Version: 1.0

## Scope: Dashboard, Predictions, Event Detail, Data Panels

## Design System: Dark Tactical Intelligence Interface

---

# 1. 🎯 Design Vision

WatchOver is positioned as:

> A real-time geopolitical intelligence & forecasting interface combining live event tracking, predictive markets, and data analytics.

The UI language communicates:

* Authority
* Precision
* Real-time intelligence
* Tactical awareness
* Financial terminal aesthetic (Bloomberg / Palantir / trading terminal influence)

The design must feel:

* Live
* Mission-critical
* Analytical
* Data-dense but controlled

---

# 2. 🧠 Core Design Principles

### 2.1 Tactical Minimalism

* Black / near-black background
* Thin separators
* Sparse but intentional color
* High-contrast signal colors

### 2.2 Signal-Driven Color System

Color communicates meaning, never decoration.

| Meaning          | Color  |
| ---------------- | ------ |
| Escalation       | Red    |
| De-escalation    | Green  |
| Neutral / Medium | Yellow |
| Informational    | Blue   |
| Warning          | Amber  |

---

# 3. 🏗 Layout Architecture

## 3.1 Global Layout Grid

Structure (Desktop ≥ 1280px):

```
-------------------------------------------------
| Top Navigation Bar                            |
-------------------------------------------------
| Left Panel | Center Globe | Right Panel       |
-------------------------------------------------
| Bottom Tension Graph                          |
-------------------------------------------------
```

### Column Distribution

* Left Panel: 22–25%
* Center Globe: 45–50%
* Right Panel: 25–30%

Responsive behavior:

* <1024px → right panel collapses
* <768px → stacked layout

| Level | Size    |

# 16. Accessibility

* Color + icon (not color alone)
* Keyboard navigable
* Tooltip readable
* Contrast ≥ WCAG AA
* Focus states visible

---

# 17. Performance Considerations

* Globe must remain 60fps
* Lazy load right panel data
* Virtualize feed list
* WebSocket for live events
* Debounced search

---

# 18. Motion Design Language

* Subtle
* Linear
* No playful easing
* Slight fade transitions
* Slow global rotation

---

# 19. Responsive Strategy

Desktop: Full 3-panel
Tablet: Collapse right panel
Mobile:

* Globe fullscreen
* Feed slide-up
* Predictions separate page
